# React Redux Starter

> Another React boilerplate for Telstra Homesite with Redux + Webpack + Babel

## Start

```sh
$ npm install
$ npm start
```

You can change server port using following command.

```sh
$ PORT=9000 npm start
```

## Testing

```sh
$ npm test
```
